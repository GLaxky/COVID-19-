# COVID-19背景下的网络社会心态及公众情绪分析
——数据科学大作业的代码和文本数据库——by李晓康-191250075、陈伟豪-191250010、李若璇-191250072

我们爬取疫情期间的新闻以及网友们的或有趣或无奈的评论，并使用数据科学和机器学习等方法分析了这一时间段网友的心态和情绪变化，最后做出了心理预测。 【研究成果见本仓库中的研究报告.pdf】
